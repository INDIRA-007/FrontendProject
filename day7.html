transisition -> to change the way an element is displayed on the page over a period of time transition: all 0.5s ease; /* all properties will change over a period transition: delay, duration, timing-function */ transform -> to change the position, size, and rotation of an element translateX(50px) translateY(50px); /* move the element 50px to the right and 50px down */ rotate(45deg); /* rotate the element 45 degrees */ skewX(20deg) skewY(20deg); /* skew the element 20 degrees in the X and Y direction */ scale(1.5); /* scale the element to 1.5 times its original size */ using while hovering the element border radisus -> to change the shape of the border of an element border-radius: 10px; /* round the corners of the element */ border -radius: 50%; /* make the element a circle */ box model css border, margin, padding content border - the outermost layer of the box model, it is used to create a border around the element margin - the space outside the border, it is used to create space between elements padding - the space inside the border, it is used to create space between the content and the border content - the innermost layer of the box model, it is used to display the content of the element box-shadow -> to add a shadow to an element box-shadow: 10px 10px 5px rgba(0,0,0,0.5); /* add a shadow to the element */
